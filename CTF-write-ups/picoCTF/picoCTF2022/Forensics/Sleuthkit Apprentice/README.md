## Solution

* AccessData FTK (Forensic Toolkit) 是一種專業的數位取證工具，用於從電腦和移動設備中收集和分析數據。它可以用於調查犯罪、法律訴訟和其他數位取證工作。FTK 可以讀取和顯示各種文件格式，並提供豐富的分析功能，如文件類型分析、檔案關聯分析、磁碟鏡像和密碼破解。
